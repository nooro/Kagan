This is an Isometric level editor for the game Blank.

P for adding new layer
O for removing the last layer
L for changing the layer ++
K for changing the layer --

- for zoom out
+ for zoom in 

arrow keys for camera movement

Delete for disabling the hovered tile
LeftMouseButton for placing your selected tile on the hovered tile

Menu with tiles (hardcored) resolution 1367:768 expect a fix

Click S to save the map

There is no function for loading a map ... yet
